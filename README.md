## 🚀 Quick Start

### Prerequisites
- ✅ **Java JDK 11+** ([Download](https://www.oracle.com/java/technologies/downloads/))
- ✅ **IDE** (IntelliJ IDEA, Eclipse, or VS Code)
- ✅ **Git** for version control

### Setup Instructions

1. **Clone the repository:**
   ```bash
   git clone https://github.com/amitbirajdar0311/CoreJava.git
   cd CoreJava
   ```

2. **Compile the project:**
   ```bash
   javac -d bin src/main/java/com/ab/*/*.java
   ```

3. **Run a module:**
   ```bash
   java -cp bin com.ab.Main
   ```

## 💡 Key Concepts Overview

### Programming Fundamentals
| Concept | Purpose |
|---------|----------|
| **Variables** | Store data in memory |
| **Data Types** | Define variable memory size and type |
| **Operators** | Perform operations on variables |
| **Conditionals** | Make decisions in code |
| **Loops** | Repeat code execution |

### Intermediate Concepts
| Concept | Purpose |
|---------|----------|
| **Functions** | Organize and reuse code |
| **Arrays** | Store multiple elements |
| **Strings** | Work with text |
| **Recursion** | Function calling itself |

### Advanced Concepts
| Concept | Purpose |
|---------|----------|
| **OOP** | Organize code into objects |
| **Inheritance** | Reuse code from parent class |
| **Polymorphism** | Same method, different behavior |
| **Multithreading** | Run multiple tasks simultaneously |
| **Collections** | Dynamic data structures |

## 🛠️ Compilation & Execution

### Using Command Line
```bash
# Compile all Java files
javac -d bin src/main/java/com/ab/*.java

# Run specific class
java -cp bin com.ab.Main

# Compile single file
javac -d bin src/main/java/com/ab/_02VariableAndDataType/Example.java
```

### Using Maven
```bash
# Compile
mvn compile

# Run tests
mvn test

# Package
mvn package
```

### Using IDE (IntelliJ IDEA)
1. Open project
2. Right-click on file → Run
3. Or press `Ctrl+Shift+F10` (Windows) / `Ctrl+Shift+R` (Mac)
