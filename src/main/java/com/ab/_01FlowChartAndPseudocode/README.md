# Module 01: Flow Chart & Pseudocode - README

## 📚 Overview

This module teaches you how to approach problems systematically using flowcharts and pseudocode before writing actual code.

## 🎯 Learning Objectives

- ✅ Understand flowchart symbols and their meanings
- ✅ Create flowcharts for simple problems
- ✅ Write effective pseudocode
- ✅ Convert flowcharts to pseudocode and vice versa
- ✅ Solve problems systematically

---

## 📋 Topics Covered

### Flowchart Symbols
- **Oval/Terminal**: Start/End of program
- **Rectangle**: Process/Calculation
- **Diamond**: Decision (if/else)
- **Parallelogram**: Input/Output
- **Arrow**: Flow of control

### Pseudocode Writing
- Linear pseudocode
- Conditional pseudocode
- Loop-based pseudocode
- Function pseudocode

---

## 💻 Example Problems

### Problem 1: Find Maximum of Two Numbers
```
Flowchart:
  START
    ↓
  INPUT: a, b
    ↓
  a > b?
    ├─ YES → OUTPUT: a
    └─ NO → OUTPUT: b
    ↓
  END
```

### Problem 2: Sum of First N Numbers
```
Pseudocode:
START
  READ n
  sum = 0
  i = 1
  WHILE i <= n DO
    sum = sum + i
    i = i + 1
  END WHILE
  PRINT sum
END
```
